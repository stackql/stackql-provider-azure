<TabItem value="list">

List all secrets of a HealthBot.

```sql
SELECT
secrets
FROM azure_extras.health_bot.bots_secrets
WHERE subscriptionId = '{{ subscriptionId }}' -- required
AND resourceGroupName = '{{ resourceGroupName }}' -- required
AND botName = '{{ botName }}' -- required
;
```
