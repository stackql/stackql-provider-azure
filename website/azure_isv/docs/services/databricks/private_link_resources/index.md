<TabItem value="get">

Get the specified private link resource for the given group id (sub-resource)

```sql
SELECT
id,
name,
properties,
systemData,
type
FROM azure_isv.databricks.private_link_resources
WHERE resourceGroupName = '{{ resourceGroupName }}' -- required
AND workspaceName = '{{ workspaceName }}' -- required
AND subscriptionId = '{{ subscriptionId }}' -- required
AND groupId = '{{ groupId }}' -- required
;
```
</TabItem>
<TabItem value="list">

List private link resources for a given workspace

```sql
SELECT
id,
name,
properties,
systemData,
type
FROM azure_isv.databricks.private_link_resources
WHERE resourceGroupName = '{{ resourceGroupName }}' -- required
AND workspaceName = '{{ workspaceName }}' -- required
AND subscriptionId = '{{ subscriptionId }}' -- required
;
```
