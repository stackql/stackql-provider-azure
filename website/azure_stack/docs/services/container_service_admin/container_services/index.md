Returns a list of managed clusters present in all tenant location.

```sql
SELECT
id,
name,
type
FROM azure_stack.container_service_admin.container_services
WHERE subscriptionId = '{{ subscriptionId }}' -- required
AND location = '{{ location }}' -- required
;
```
