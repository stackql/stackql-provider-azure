## `SELECT` examples

<Tabs
    defaultValue="get"
    values={[
        { label: 'get', value: 'get' },
        { label: 'list', value: 'list' }
    ]}
>
<TabItem value="get">

Returns the requested logical subnet.

```sql
SELECT
id,
name,
properties,
type
FROM azure_stack.fabric_admin.logical_subnets
WHERE subscriptionId = '{{ subscriptionId }}' -- required
AND resourceGroupName = '{{ resourceGroupName }}' -- required
AND location = '{{ location }}' -- required
AND logicalNetwork = '{{ logicalNetwork }}' -- required
AND logicalSubnet = '{{ logicalSubnet }}' -- required
;
```
</TabItem>
<TabItem value="list">

Returns a list of all logical subnets.

```sql
SELECT
id,
name,
properties,
type
FROM azure_stack.fabric_admin.logical_subnets
WHERE subscriptionId = '{{ subscriptionId }}' -- required
AND resourceGroupName = '{{ resourceGroupName }}' -- required
AND location = '{{ location }}' -- required
AND logicalNetwork = '{{ logicalNetwork }}' -- required
AND $filter = '{{ $filter }}'
;
```
</TabItem>
</Tabs>
