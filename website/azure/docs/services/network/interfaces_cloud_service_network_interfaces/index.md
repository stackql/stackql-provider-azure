<TabItem value="get">

Get the specified network interface in a cloud service.

```sql
SELECT
id,
name,
etag,
extendedLocation,
properties,
systemData,
type
FROM azure.network.interfaces_cloud_service_network_interfaces
WHERE resourceGroupName = '{{ resourceGroupName }}' -- required
AND cloudServiceName = '{{ cloudServiceName }}' -- required
AND roleInstanceName = '{{ roleInstanceName }}' -- required
AND networkInterfaceName = '{{ networkInterfaceName }}' -- required
AND subscriptionId = '{{ subscriptionId }}' -- required
AND api-version = '{{ api-version }}'
AND $expand = '{{ $expand }}'
;
```
</TabItem>
<TabItem value="list">
