## `SELECT` examples

<Tabs
    defaultValue="get"
    values={[
        { label: 'get', value: 'get' },
        { label: 'list_by_firmware', value: 'list_by_firmware' }
    ]}
>
<TabItem value="get">

Get an analysis result summary of a firmware by name.

```sql
SELECT
id,
name,
properties,
systemData,
type
FROM azure.iot_firmware_defense.summaries
WHERE subscriptionId = '{{ subscriptionId }}' -- required
AND resourceGroupName = '{{ resourceGroupName }}' -- required
AND workspaceName = '{{ workspaceName }}' -- required
AND firmwareId = '{{ firmwareId }}' -- required
AND summaryName = '{{ summaryName }}' -- required
;
```
</TabItem>
<TabItem value="list_by_firmware">

Lists analysis result summary names of a firmware. To fetch the full summary data, get that summary by name.

```sql
SELECT
id,
name,
properties,
systemData,
type
FROM azure.iot_firmware_defense.summaries
WHERE subscriptionId = '{{ subscriptionId }}' -- required
AND resourceGroupName = '{{ resourceGroupName }}' -- required
AND workspaceName = '{{ workspaceName }}' -- required
AND firmwareId = '{{ firmwareId }}' -- required
;
```
