Lists available Disk Pool Skus in an Azure location.

```sql
SELECT
additionalCapabilities,
availabilityZones,
sku
FROM azure.storage_pool.disk_pool_zones
WHERE subscriptionId = '{{ subscriptionId }}' -- required
AND location = '{{ location }}' -- required
;
```
