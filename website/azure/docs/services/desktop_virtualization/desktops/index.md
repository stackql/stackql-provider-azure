## `SELECT` examples

<Tabs
    defaultValue="get"
    values={[
        { label: 'get', value: 'get' },
        { label: 'list', value: 'list' }
    ]}
>
<TabItem value="get">

Get a desktop.

```sql
SELECT
id,
name,
properties,
systemData,
type
FROM azure.desktop_virtualization.desktops
WHERE subscriptionId = '{{ subscriptionId }}' -- required
AND resourceGroupName = '{{ resourceGroupName }}' -- required
AND applicationGroupName = '{{ applicationGroupName }}' -- required
AND desktopName = '{{ desktopName }}' -- required
;
```
</TabItem>

<Tabs

<TabItem value="update">

Update a desktop.

```sql
UPDATE azure.desktop_virtualization.desktops
SET 
data__tags = '{{ tags }}',
data__properties = '{{ properties }}'
WHERE 
subscriptionId = '{{ subscriptionId }}' --required
AND resourceGroupName = '{{ resourceGroupName }}' --required
AND applicationGroupName = '{{ applicationGroupName }}' --required
AND desktopName = '{{ desktopName }}' --required
RETURNING
id,
name,
properties,
systemData,
type;
```
