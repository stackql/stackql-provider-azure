Lists the usages for the specified location.

```sql
SELECT
name,
currentValue,
limit,
unit
FROM azure.hdinsight.locations_usages
WHERE subscriptionId = '{{ subscriptionId }}' -- required
AND location = '{{ location }}' -- required
;
```
