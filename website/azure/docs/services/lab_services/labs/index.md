Operation to create or update a lab resource.

```sql
INSERT INTO azure.lab_services.labs (
data__tags,
data__location,
data__properties
)

'{{ tags }}',
'{{ location }}',
'{{ properties }}' /* required */
RETURNING
location,
properties,
systemData,
tags
;
```
