Get information about a configuration profile

```sql
SELECT
location,
properties,
systemData,
tags
FROM azure.automanage.configuration_profiles
WHERE configurationProfileName = '{{ configurationProfileName }}' -- required
AND subscriptionId = '{{ subscriptionId }}' -- required
AND resourceGroupName = '{{ resourceGroupName }}' -- required
;
```

Updates a configuration profile

```sql
UPDATE azure.automanage.configuration_profiles
SET 
data__properties = '{{ properties }}',
data__tags = '{{ tags }}'
WHERE 
configurationProfileName = '{{ configurationProfileName }}' --required
AND subscriptionId = '{{ subscriptionId }}' --required

RETURNING
location,
properties,
systemData,
tags;
```
