## `SELECT` examples

<Tabs
    defaultValue="list"
    values={[
        { label: 'list', value: 'list' }
    ]}
>
<TabItem value="list">

Lists the operations available from this provider.

```sql
SELECT
name,
display,
isDataAction,
origin,
properties
FROM azure.app_configuration.operations
WHERE $skipToken = '{{ $skipToken }}'
;
```
</TabItem>
</Tabs>

<TabItem value="check_name_availability">

Checks whether the configuration store name is available for use.

```sql
EXEC azure.app_configuration.operations.check_name_availability 
@subscriptionId='{{ subscriptionId }}' --required 
@@json=
'{
"name": "{{ name }}", 
"type": "{{ type }}"
}'
;
```
</TabItem>
<TabItem value="regional_check_name_availability">

Checks whether the configuration store name is available for use.

```sql
EXEC azure.app_configuration.operations.regional_check_name_availability 
@subscriptionId='{{ subscriptionId }}' --required, 
@location='{{ location }}' --required 
@@json=
'{
"name": "{{ name }}", 
"type": "{{ type }}"
}'
;
```
</TabItem>
