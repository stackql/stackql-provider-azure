Retrieves the subscription's current quota information in a particular region.

```sql
SELECT
id,
name,
properties,
type
FROM azure.stream_analytics.subscriptions_quotas
WHERE location = '{{ location }}' -- required
AND subscriptionId = '{{ subscriptionId }}' -- required
;
```
