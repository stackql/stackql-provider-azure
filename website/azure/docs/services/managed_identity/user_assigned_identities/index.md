<Tabs

Update an identity in the specified subscription and resource group.

```sql
UPDATE azure.managed_identity.user_assigned_identities
SET 
data__location = '{{ location }}',
data__tags = '{{ tags }}'
WHERE 
subscriptionId = '{{ subscriptionId }}' --required

RETURNING
location,
properties,
tags;
```
